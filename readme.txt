Cara Menjalankan i-Posyandu


1. Projek ini dibuat dengan IDE IntelliJ
2. Proyek ini menggunakan bahasa pemrograman Java
3. Untuk mengakses main drive / main class, yaitu dapat mengakses pada folder src -> posyandu -> app -> PosyanduApp.java
4. Karena sering terjadinya issue/error yang terjadi ketika proyek diakses dengan IDE atau teks editor lain
   Jika proyek ini juga error saat dibuka dengan IDE ataupun teks editor lain, misalnya VSCode.
   Harapannya dapat membuka proyek ini dengan IDE IntelliJ atau mengaksesnya melalui terminal
5. Terima kasih atas kesempatan dan pengertiannya
